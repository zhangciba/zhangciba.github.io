Linux SSH 远程登录后显示-bash-4.1$的解决办法

发表于 2016-06-30 | 分类于 ssh

linux系统新建的用户用ssh远程登陆显示-bash-4.1$，不显示用户名路径。

产生这种问题有三种原因。

  • 没有给用户创建/home/username主目录
  • 用户没有/home/username主目录的读取权限
  • 主目录/home/username下面缺少shell环境

解决方法

  • 创建主目录

  • 修改用户的所有者和组

  • 拷贝Shell环境到主目录下
